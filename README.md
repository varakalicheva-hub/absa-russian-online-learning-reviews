# ABSA for Russian Online Course Reviews

Repository accompanying a SPECOM paper on aspect-based sentiment analysis (ABSA) of Russian-language online course reviews using large language models (LLMs).

## Overview

- **Task:** Aspect-Based Sentiment Analysis (ABSA)
- **Language:** Russian
- **Domain:** Online Education Reviews
- **Annotation Format:** Aspect Term – Aspect Category – Opinion – Sentiment
- **Models:** GPT-4.1 mini, DeepSeek V3.2, YandexGPT 5.1 Pro

## Repository Contents

This repository contains:

- a corpus of Russian-language online course reviews;
- a gold-standard annotation;
- LLM-generated annotations;
- notebooks for automatic annotation;
- notebooks for annotation quality evaluation;
- prompt templates used for LLM-based annotation.

## Data

The `data/` directory contains the corpus and all annotation files.

### Corpus Files

| File | Description |
|------|-------------|
| `01_full_corpus_2400.csv` | Complete corpus containing all collected reviews (2,400 reviews) |
| `02_annotation_sample_300.csv` | Sample used for automatic annotation (300 reviews) |
| `03_evaluation_sample_100.csv` | Sample used for evaluation (100 reviews) |

### Annotation Files

| File | Description |
|------|-------------|
| `04_gold_annotation_100.csv` | Gold-standard annotation |
| `05_gpt41mini_annotation_100.csv` | Annotation generated by GPT-4.1 mini |
| `06_deepseek_v32_annotation_100.csv` | Annotation generated by DeepSeek V3.2 |
| `07_yandexgpt_51pro_annotation_100.csv` | Annotation generated by YandexGPT 5.1 Pro |

Files `04`–`07` contain annotations for the same set of 100 reviews, allowing direct comparison between expert annotation and LLM-generated annotations.

### Data Format

#### Unannotated Files

| Column | Description |
|----------|-------------|
| `review_id` | Unique review identifier |
| `review_text` | Review text |
| `review_url` | Source URL |

#### Annotated Files

| Column | Description |
|----------|-------------|
| `review_id` | Unique review identifier |
| `review_text` | Review text |
| `aspect_term` | Aspect mention extracted from the review |
| `opinion` | Opinion expression associated with the aspect |
| `aspect` | Aspect category |
| `sentiment` | Sentiment polarity (positive, negative, neutral) |
| `review_url` | Source URL |

## Code

The `notebooks/` directory contains Jupyter notebooks developed in Google Colab.

| Notebook | Purpose |
|-----------|---------|
| `llm_annotation.ipynb` | Automatic annotation using LLM APIs |
| `quadruple_matching_evaluation.ipynb` | Evaluation based on exact and partial quadruple matching (precision, recall, F1-score) |
| `semantic_similarity_evaluation.ipynb` | Evaluation based on semantic similarity of extracted spans |

### Automatic Annotation

The annotation pipeline uses LLM APIs to extract quadruples in the following format:

> Aspect Term – Aspect Category – Opinion – Sentiment

Prompt templates used for annotation are included in the notebook `llm_annotation.ipynb`.

---

## Evaluation Methodology

Automatic annotations are evaluated against the gold-standard annotation using two complementary approaches.

### Exact and Partial Matching

Predicted and gold-standard quadruples are compared using strict and partial matching criteria. Precision, recall, and F1-score are calculated for both settings.

### Semantic Similarity

Cosine similarity is computed between vector representations of automatically extracted aspect terms and opinion expressions and their corresponding gold-standard annotations.
